# Kessoku the Shot 📱
> 专注于**算法竞赛直播**的移动机位推流 app

## 🛠️ 技术栈
- vue 3.2
- vue-class-component
- capacitor
- typescript
- element-plus

## 📦️ 环境准备
- nodeJS >= 22
- pnpm >= 8

## 🚀 快速开始

### 依赖安装
```bash
pnpm i
```

### 启动开发
在虚拟机中安装软件（以 IOS 模拟器为例）
```bash
npx cap run ios
```

随后启动 vite 服务器
```bash
pnpm dev
```

随后在模拟器中打开软件即可启动开发
